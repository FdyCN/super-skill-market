## 如何贡献

**方式一：提 Pull Request**

1. Fork 本仓库
2. 在 README.md 对应分类的表格末尾加一行
3. 确保链接指向原始仓库（非 fork）
4. 提 PR，标题格式：`add: skill-name`

**方式二：提 Issue 提名**

在 [Issues](../../issues) 中按如下格式提名，维护者会录入：

```
Skill 名：xxx
分类：人物蒸馏 / 脑洞大开（子类：技术 / 经济&投资 / 哲学&智慧 / 商业&管理）
一句话简介：xxx
原始仓库链接：https://github.com/xxx/yyy
```
