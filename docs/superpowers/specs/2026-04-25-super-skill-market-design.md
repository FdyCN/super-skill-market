# Super Skill Market — 设计文档

**日期**：2026-04-25  
**状态**：已批准

---

## 目标

搭建一个轻量化的 Claude Code Skills 精选集市，收录两类创意 skill：**人物蒸馏**（以某位人物的思维框架指导工作）和**脑洞大开**（出乎意料的非常规视角）。优先收录中文 skill。

## 定位

- **起点**：个人精品策展，自用为主
- **演进**：开放社区贡献通道，任何人可 PR 提交

## 核心原则

- 只在 README 中维护链接 + 简介，不存储 skill 文件本体
- 保持对原创仓库的尊重，每条记录必须包含原始链接
- 极简贡献门槛：提 PR 加一行表格即可

## README 结构

```
# Super Skill Market

## 人物蒸馏
### 技术
### 经济 & 投资
### 哲学 & 智慧
### 商业 & 管理

## 脑洞大开

## 如何贡献
```

### 人物分类说明

| 分类 | 收录范围 |
|------|---------|
| 技术 | 科学家、工程师、发明家（费曼、图灵、冯·诺依曼等） |
| 经济 & 投资 | 经济学家、投资人（巴菲特、芒格、凯恩斯等） |
| 哲学 & 智慧 | 哲学家、思想家、古典智慧（苏格拉底、王阳明、老子等） |
| 商业 & 管理 | 企业家、管理者（乔布斯、稻盛和夫、曾国藩等） |

### 每条记录格式

| Skill | 简介 | 仓库 |
|-------|------|------|
| `` `skill-name` `` | 一句话描述 skill 的视角和用途 | [作者/仓库名](url) |

## 贡献规则

1. Fork 本仓库
2. 在 README 对应分类的表格末尾加一行
3. 确保链接指向原始仓库（非 fork）
4. 提 Pull Request，标题格式：`add: skill-name`
